MindCare Hospital is a modern mental health care web application designed to help users discover expert doctors, book appointments, and access mental health support services seamlessly.

##  Features

- Clean and modern Home page
- Expert listing with profile details
- Appointment booking flow
- Admin panel for managing experts also add,edit and delete experts
- Responsive UI using Tailwind CSS
- Built with React + Vite

## Tech Stack

- **Frontend:** React, Vite
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Version Control:** Git & GitHub

## Clear Setup and Run Instructions
**1.Clone the repository**
git clone https://github.com/stebinsha/mindcare-hospital.git
cd mindcare-hospital
**2.Install dependencies**
npm install
npm install lucide-react
npm install react-router-dom
**3.Run the development server**
npm run dev


## Assumptions Made
1.The project uses only frontend; no backend or API integration.
2.Admin can add, edit, or delete doctors; Viewer can only view.
3.All data is stored in localStorage.
4.Default admin and viewer credentials are stored in .env for local development.
5.Images can be URLs; placeholders are used if empty.
